# Triasha Sarkar — Final Portfolio Website

Static research portfolio for Triasha Sarkar.

## Profile direction

**Sustainable transportation · Mobility resilience · Scientific ML · Digital twins · Uncertainty quantification · Surrogate modeling**

The site presents Triasha's work as a coherent research profile across sustainable transportation systems, aviation decarbonization, mobility resilience, scientific machine learning, uncertainty-aware modeling, and digital twins.

## Files

- `index.html` — homepage
- `case-green-tea.html` — GREEN TEA / forest-residue SAF case study
- `case-atlanta.html` — Atlanta Mobility Resilience Digital Twin case study
- `case-scientific-ml.html` — NURBS BEM + surrogate modeling projects
- `case-srm.html` — SRM time-series ML case study
- `styles.css` — visual system
- `script.js` — menu + reveal interactions
- `assets/` — photos and SVG graphics

## Deploying to GitHub Pages

Upload the **contents inside this folder** to the root of the GitHub `Portfolio` repository.

The repository root should show:

```text
index.html
case-green-tea.html
case-atlanta.html
case-scientific-ml.html
case-srm.html
styles.css
script.js
assets/
README.md
```

Then commit and wait for GitHub Pages to deploy.

Live URL format:

```text
https://triasha72.github.io/Portfolio/
```
