# Triasha Sarkar — Harsha-inspired research portfolio website

This is a static, multi-page portfolio website inspired by the large-type, visual case-study feel of the reference portfolio, but rewritten for an academic/research review context.

## Files
- `index.html` — homepage
- `case-green-tea.html` — GREEN TEA / SAF case study
- `case-atlanta.html` — Atlanta Mobility Resilience Digital Twin case study
- `case-srm.html` — AE8900/SMAC writing-sample companion
- `styles.css` — full responsive visual design
- `script.js` — mobile menu, reveal interactions, cursor effect
- `assets/` — editable SVG placeholders

## What to replace before sharing with SCaRP
1. Replace `assets/green-tea-system.svg` with your actual GREEN TEA system boundary / emissions figure.
2. Replace `assets/atlanta-map.svg` with your actual Atlanta mobility/accessibility map or screenshot.
3. Replace `assets/srm-curves.svg` with your actual AE8900 thrust-time/clustering figure.
4. Update any project wording that you want to make more precise before submission.

## Deployment
This folder can be uploaded directly to GitHub Pages, Netlify, Vercel, or any static web host.
