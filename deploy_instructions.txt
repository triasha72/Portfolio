DEPLOYMENT INSTRUCTIONS

Option 1: GitHub Pages
1. Create a new public GitHub repository, e.g. `triasha-portfolio`.
2. Upload all files in this folder to the repository root.
3. Go to Settings → Pages.
4. Under Build and deployment, choose Deploy from branch.
5. Select the `main` branch and `/root` folder.
6. Save. GitHub will generate a public link.

Option 2: Netlify
1. Go to netlify.com and choose Add new site → Deploy manually.
2. Drag and drop this entire folder.
3. Netlify will create a public portfolio URL.

Option 3: Local preview
Double-click `index.html` or open it in a browser.
