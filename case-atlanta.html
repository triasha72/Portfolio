Atlanta Mobility Resilience Digital Twin

A geospatial mobility simulator focused on how road-network disruptions alter travel time, accessibility, and equity across Atlanta.

← Back to portfolio
Role
Independent project owner
Methods
Network analysis, accessibility metrics, disruption scenarios, equity-sensitive interpretation
Tools
Python, geospatial libraries, Jupyter, GitHub
Status
Public GitHub project; add screenshots/maps before sharing

Why it matters

Transportation systems fail unevenly. A disruption that looks small in network terms can create major access burdens for specific neighborhoods, trips, or communities.

What I contributed

  • Structured the project around disruption scenarios rather than static network description.
  • Focused the output on travel-time, accessibility, and equity impacts.
  • Built a project narrative that directly connects engineering-style simulation to planning questions.

Planning bridge

This project gives a planning committee an immediate signal that your quantitative background can move into urban mobility research. It also pairs naturally with future work in spatial analysis, GIS, and travel behavior.
